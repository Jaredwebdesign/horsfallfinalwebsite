About

This website is about web design created by Jared Horsfall as a student project for ARTD 245: Web Design at North Central College in Naperville, IL, in 2023. All the text comes from the following articles located on the Dribbble website. All the images were created by me using Photoshop. The website has been created using HTML and CSS in Dreamweaver.

Featured Articles

  1. Choosing typography for web design: 5 things you need to consider
  2. Choosing colors for web design: A practical UI color application guide
  3. 11 Principles of Design (and How to Use Them)

Project Description

Design and develop a visually pleasing, interactive website with multiple HTML pages, navigation connecting all the pages, featuring a cohesive color scheme, a clear visual hierarchy, a balanced layout, exciting visuals, and text. The goal is to learn how to code/develop an interactive website by applying design principles.
